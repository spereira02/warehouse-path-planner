## System Demonstration

The following video shows the full warehouse simulation in which the planner was originally used.

The system includes:
- multi-robot task allocation
- waypoint routing
- collision-aware navigation
- low-level motor control for differential-drive robots

The planner published in this repository corresponds to the **path planning component** used to generate collision-free routes between waypoints.
